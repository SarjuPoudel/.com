Financial Aid Application: Cost and Fee Waivers

Just like College Application, Financial Aid Application may also incurred some cost on you. Colleges accepting paper application like ISFAA or College's own financial aid application are free unless otherwise stated in college's website. The main financial aid application cost during college application is associated with colleges taking CSS profile to evaluate your financial situation. Initial application to CSS cost $25 and $16 dollars for each additional application. During your college application period, there are certain ways to minimize overal CSS Profile cost which as described below.

    Ways to Reduce Cost of CSS Profile

  • Request a CSS Profile Fee Wavier Code.
  • Request if your college accepts a paper CSS profile application
  • Request if your college accepts ISFAA forms, instead.
  • Ask if there are any other alternative forms provided by your college to replace CSS Profile Submission.
  • Attain Online Webinars hosted by colleges, where they like offer Fee Waiver Code.
  • Reconsider your college list, if your total application fees are beyond your estimated budget.
  • If your are Opportunity Fund Participants of USEF Nepal, check with your advisors if there are any alternatives.

While making a request for fee waivers or alternative approaches, your communication and negotiation skill plays an important role. Check out more information on how to communicate effectively with your college.

Note: CSS Fee waiver codes are specific to colleges which you received from. It will cover either $25 or $16 depending on if it is your first time submission to any college.
